# 🧠 Brain Tumor Detection using CNN

A deep learning web application built using PyTorch and Streamlit to classify MRI brain scans as Tumor or No Tumor.

## Features

- CNN-based classification
- 92.16% test accuracy
- Streamlit web application
- Confidence score prediction
- Professional healthcare dashboard

## Technologies

- Python
- PyTorch
- Streamlit
- Torchvision
- PIL
